What changed in v0.1.9

Web SPA + PWA

Legacy web pages replaced by a responsive SPA with modern routes.

Watching Now

Series watch-state, resume flows, and richer episode navigation.

DRM-Aware Playback

Expanded player payload with DRM fields and better web handling.

FFmpeg HLS Path

Optional in-memory transmuxing flow for TS-style browser playback.

Playlist Publishing

Published exports available via /iptv.m3u and /iptv.m3u8.

Import Improvements

Stronger parsing for Stalker/Xtream/M3U and improved account tools.

Capabilities

Built for real IPTV workflows

Multi-Protocol Core

Stalker Portal, Xtream Codes, M3U (local/remote), and RSS including YouTube feeds.

Player Flexibility

Embedded VLC/Lite, up to 3 external players, and browser-player routing options.

Smart Filtering

Category and channel filtering with pause toggle, also respected by web routes.

Network Streaming

Run the built-in HTTP server and access channels from phones, tablets, and TVs.

Bookmarks + Ordering

Favorites, categories, and playlist-style export endpoints for remote players.

Cache Controls

Cache expiry, global clear/reload, account-level reload, and watching-now reset.

Import Modes

Fast onboarding for large account sets

Stalker Portal

MAC-based import with optional serial/device/signature fields, timezone and HTTP method support.

Xtream Codes

Labeled and unlabeled username/password parsing with automatic unique account naming.

M3U / M3U8

Bulk playlist import with optional M3U-to-Xtream conversion for compatible URLs.

Open M3U Guide

Compatibility

System requirements

Current published packages: Windows x86_64, Linux x86_64, macOS x86_64 + aarch64.

  • Java 25 bundled in installers
  • VLC recommended for best embedded playback
  • yt-dlp required for YouTube RSS playback
  • FFmpeg required for TS-oriented web compatibility mode
