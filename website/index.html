Key Features

Multi-Protocol Support

Stalker Portal, Xtream Codes, M3U Playlists (Local & Remote), and RSS Feeds.

Advanced Player Support

Includes embedded VLC & Lite players. Fully supports external players like MPV or SMPlayer.

YouTube Integration

Watch YouTube channels directly via RSS feeds (requires yt-dlp).

Web Server & Headless

Stream to any device on your network via browser. Supports headless mode for server-only usage.

Search & Favorites

Real-time search filtering and easy favorites management for quick access to your content.

Smart Filtering

Exclude unwanted categories or channels. Pause filters on the fly for full access.

Customizable UI

Modern tab-based interface with Dark Mode, custom fonts, and scaling options.

Auto-Updates

Built-in update checker ensures you are always running the latest version.

Performance

SQLite caching for fast load times. Options to pause or clear cache as needed.

System Requirements

UIPTV is available for Windows (x64, ARM64), Linux (x64, ARM64), and macOS (Intel, Apple Silicon).

For the best experience, we recommend the following:

  • Java 25 Bundled with installers, no separate installation needed.
  • VLC Media Player Highly recommended for the best video playback compatibility.
  • yt-dlp Required if you intend to play YouTube videos via RSS feeds.
