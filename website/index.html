Compatibility

System requirements

Current published targets: Windows x86_64, Linux x86_64, macOS x86_64 + aarch64. Linux releases are distributed as .deb, optional .rpm, and a portable app bundle archive.

  • Java 25 bundled in installers
  • VLC recommended for best embedded playback
  • yt-dlp required for YouTube RSS playback
  • FFmpeg required for TS-oriented web compatibility mode

Ready to test it?

Grab the latest release or go straight to the setup docs.

Linux users can choose between package installs and the portable app bundle archive. Theme settings also support optional light/dark CSS overrides.
